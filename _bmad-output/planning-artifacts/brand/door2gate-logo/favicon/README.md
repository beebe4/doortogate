# Favicon Package

## Installation Instructions

1. Copy all the favicon image files to the root of your website
2. Add the contents of `favicon-snippet.html` to the <head> section of your HTML

## Files Included

- favicon.ico - Traditional favicon
- favicon-16x16.png - 16x16 favicon
- favicon-32x32.png - 32x32 favicon
- favicon-48x48.png - 48x48 favicon
- android-chrome-192x192.png - Android Chrome icon (192x192)
- android-chrome-512x512.png - Android Chrome icon (512x512)
- apple-touch-icon.png - Apple touch icon (180x180)
- favicon-snippet.html - HTML code snippet to add to your <head> tag

## Usage

Simply paste the HTML snippet into your <head> tag and you're done!
